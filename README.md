# CODTECH Task 1 - Sales Trend Visualization

## Objective
The objective of this project is to analyze retail sales data and identify monthly sales trends using data visualization techniques.

## Dataset
Retail and Warehouse Sale Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Data Cleaning
- Removed null values
- Removed duplicate records
- Cleaned column names
- Converted retail sales data into numeric format

## Analysis Performed
- Loaded and explored the dataset
- Grouped retail sales by month
- Calculated total monthly retail sales
- Created a line chart to visualize sales trends over time

## Visualization
A line graph was generated to display monthly retail sales trends, helping identify patterns and fluctuations in sales performance.

## Outcome
The analysis successfully visualized retail sales trends across different months, providing insights into sales behavior and supporting data-driven decision making.
